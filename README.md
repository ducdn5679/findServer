
## Commands
Running locally:
```bash
yarn dev
```
building:
```bash
yarn build
```
Running production (build before use):
```bash
yarn start
```

Testing:
```bash
# run all unit tests
yarn test

# run all unit tests in watch mode
yarn test:watch

# run unit tests coverage
yarn test:coverage

# run all e2e tests
yarn test:e2e

# run all e2e tests in watch mode
yarn test:e2e:watch
```
